## Description ##
#  This folder is in charge of the generation of the conventional and translationally tranformed CC equations
#  All the code related to the generation of the TTCC equation as well as the restriction of the summations
#  indices should be here. Also, all the code related to the generation of the latex notation of the equations 
#  should be here. The equations will be generated here and and the CC code will expect them to be here.
# 
# To run the code type in the command line:
# python3.6 main.py
# Instructions about the input are inside the main.py file.
